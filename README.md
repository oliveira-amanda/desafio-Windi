# desafio-windi

 
 ### - Backend

```sh
#Clonar repositório
$ git clone https://github.com/oliveira-amanda/desafio-windi.git

#Acessar pastas
$ cd desafio-windi
$ cd project
$ cd backend

#Instalar dependências
$ npm install

#Executar aplicação
$ npm start

#Acesse no browser via
http://localhost:3333 
```

 ### - Frontend Web

```sh
#Acessar pastas
$ cd desafio-windi
$ cd project
$ cd frontend

#Instalar dependências
$ npm install

#Executar aplicação
$ npm start

#Acesse no browser via
http://localhost:3000
```
